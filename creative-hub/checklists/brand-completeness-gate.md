# BRAND COMPLETENESS GATE
> QA gate para validar se o processo de branding está completo antes da compilação final
> Critério: GO se ≥ 8/10 áreas aprovadas. NO-GO se ≥ 3 áreas críticas reprovadas.

---

## Como Usar

1. Execute este gate após completar todas as 9 fases do Método de Branding
2. Para cada área, marque `[x]` se aprovada ou `[ ]` se reprovada
3. Para cada item reprovado, registre o gap no campo "Gaps Identificados"
4. Calcule o score e tome a decisão GO/NO-GO

**Executor recomendado:** @qa ou @brand-director `*compile-brandbook`

**GO automático:** ≥ 8 áreas aprovadas E 0 áreas críticas reprovadas

- < 7 áreas aprovadas no total

- 7-8 áreas aprovadas com apenas 1 área não-crítica reprovada
