## CARROSSEL: Os Arquétipos de Jung e Posicionamento de Marca

---

### SLIDE 1 / GANCHO

```
ARQUÉTIPO NÃO É
PERSONALIDADE DE MARCA.

É estrutura inconsciente de decisão.
```

*Modelo: "O que parece X é, na verdade, Y." | Pilar: P1*

---

### SLIDE 2 / CONTEXTO

```
TÍTULO: O que a maioria entende errado

Quando alguém fala "arquétipo de marca",
a maioria imagina um exercício criativo.
Escolher entre Herói e Sábio como se fosse
teste de personalidade do Instagram.

O problema começa aí.
```

---

### SLIDE 3 / SUBTEXTO

```
TÍTULO: O segundo nível

Isso não é sobre nomenclatura.
É sobre o mecanismo que governa
por que alguns posicionamentos
geram adesão automática —
e outros exigem convencimento infinito.
```

---

### SLIDE 4 / FRAMEWORK 1

```
TÍTULO: O que Jung realmente disse

O arquétipo é uma estrutura psíquica universal —
um padrão de comportamento e significado
que existe no inconsciente coletivo.

Quando uma marca ativa o arquétipo certo,
ela não precisa convencer.
Ela ressoa.
```

---

### SLIDE 5 / FRAMEWORK 2

```
TÍTULO: Como isso funciona na prática

Pensa no Sábio (Apple nos anos 80, Notion agora):
não vende produto — vende inteligência.
A promessa implícita é "você vai pensar melhor."

Pensa no Rebelde (Harley-Davidson):
não vende moto — vende rejeição de normas.
Você compra identidade, não transporte.

O produto é o veículo. O arquétipo é a razão.
```

---

### SLIDE 6 / FRAMEWORK 3

```
TÍTULO: O erro que custa mais caro

Marcas que tentam ser dois arquétipos
ao mesmo tempo acabam sendo nenhum.

Sábio + Herói = confusão de posicionamento.
Rebelde + Prestativo = contradição de valores.

A clareza do arquétipo é o que permite
que o público decida em segundos
se você é "para mim" ou não.
```

---

### SLIDE 7 / RESPIRO POÉTICO

```
"Profundidade reconhece profundidade.
O arquétipo certo não precisa se explicar."
```

*Teste: ✅ Citável isolado*

---

### SLIDE 8 / SÍNTESE

```
TÍTULO: O que muda quando você entende isso

Escolher um arquétipo não é exercício criativo.
É decisão estratégica sobre qual inconsciente
você quer ativar na cabeça do seu público.

E uma vez que você entende o mecanismo —
cada decisão de copy, visual e posicionamento
começa a fazer sentido por si só.
```

---

### SLIDE 9 / FECHAMENTO

```
TÍTULO: A pergunta que importa

Qual arquétipo a sua marca está ativando agora
— independente de qual você declarou que tem?

Eles raramente são os mesmos.
```

---

## LEGENDA

```
Passei anos achando que escolher arquétipo era
uma etapa de branding que eu "passava por cima".

Quando entendi o mecanismo por trás —
não a nomenclatura, mas a psicologia —
percebi que todas as minhas decisões de marca
estavam ativando o arquétipo errado.

Qual arquétipo a sua marca está ativando hoje?
```
