## CARROSSEL: O Padrão que Repete — Criatividade e o Kármico 333

---

### SLIDE 1 / GANCHO

```
COMECEI ESSE PROJETO
3 VEZES.

Na 4ª, a diferença
não foi motivação.
```

*Modelo: Confissão + reviravolta | Pilar: P3*

---

### SLIDE 2 / CONTEXTO

```
TÍTULO: O ciclo que você conhece

A explosão de energia no começo.
A clareza de que "dessa vez é diferente."
A semana 2, a semana 4 — e o silêncio.

Não é falta de talento.
Não é sabotagem consciente.
É um padrão que se repete — e você ainda não nomeou.
```

---

### SLIDE 3 / SUBTEXTO

```
TÍTULO: O que está acontecendo

Isso não é sobre criatividade.
É sobre o padrão que governa quando você cria
e quando você para.

A maioria chama de preguiça ou procrastinação.
Eu chamo de arquitetura invisível —
e a primeira coisa a fazer é torná-la visível.
```

---

### SLIDE 4 / FRAMEWORK 1

```
TÍTULO: O kármico 333 na criação

Numerologicamente, o 333 representa o desafio
de completar o que foi começado —
a tensão entre explosão criativa e consistência de execução.

Na prática: você começa 3 projetos diferentes
antes de terminar 1.
O problema não é falta de ideias.
É que o modelo mental de criação ainda depende de estado.
```

---

### SLIDE 5 / FRAMEWORK 2

```
TÍTULO: O que muda quando você nomeia

Nomear o padrão não o elimina.
Mas transforma de falha moral em dado diagnóstico.

"Por que eu nunca termino nada" vira:
"Meu ciclo de criação colapsa no ponto X —
o que preciso ter em vigor antes desse ponto?"

A diferença entre culpa e engenharia
é onde você coloca a pergunta.
```

---

### SLIDE 6 / FRAMEWORK 3

```
TÍTULO: A 4ª tentativa

Na 4ª vez que comecei o mesmo projeto,
a diferença não foi motivação nem disciplina.

Foi que eu havia construído:
→ Um critério para quando "pronto" — antes de começar
→ Um processo mínimo que não dependia de inspiração
→ Uma definição do que era terminar (não perfeitar)

O padrão kármico não desaparece.
Você constrói estrutura ao redor dele.
```

---

### SLIDE 7 / RESPIRO POÉTICO

```
"Criatividade sem estrutura
é potencial sem destino."
```

*Teste: ✅ Citável isolado*

---

### SLIDE 8 / SÍNTESE

```
TÍTULO: O que muda na prática

Criativos com padrão de abandono não são menos talentosos.
São pessoas que ainda estão operando com o modelo errado —
esperando que o estado interno mude antes de agir.

O insight que muda o jogo:
o sistema não depende de como você está.
É exatamente para isso que ele existe.
```

---

### SLIDE 9 / FECHAMENTO

```
TÍTULO: Uma pergunta que vale

Qual projeto você começou mais de uma vez
sem conseguir levar até o fim?

Qual parte do processo você ainda
não transformou em estrutura?
```

---

## LEGENDA

```
Comecei o mesmo projeto 3 vezes antes de entender
que o problema não era motivação — era modelo.

A 4ª vez funcionou porque eu parei de tentar mudar
como eu me sentia e comecei a mudar o que estava em vigor.

Qual projeto você tem repetido no modo reinício?
```
