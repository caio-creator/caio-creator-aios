## CARROSSEL: O que aprendi construindo meu próprio brand book

---

### SLIDE 1 / GANCHO

```
ESSA SEMANA CONSTRUÍ
MEU PRÓPRIO BRAND BOOK.

O que aprendi
não estava no plano.
```

*Modelo: Bastidores de processo com revelação | Pilar: P5*

---

### SLIDE 2 / CONTEXTO

```
TÍTULO: O que é um brand book, de fato

Não é um documento de marca.
É uma série de decisões forçadas
que você não consegue adiar
quando o processo é estruturado.

Passei anos trabalhando marca de outras pessoas.
Quando sentei para fazer a minha,
a primeira coisa que descobri foi o que eu estava evitando.
```

---

### SLIDE 3 / SUBTEXTO

```
TÍTULO: O que o processo revela

Isso não é sobre ter um documento.
É sobre o que você descobre quando
é obrigado a ser preciso sobre si mesmo.

Posicionamento vago parece profundo
até você tentar escrever em uma linha
quem exatamente você atende — e por que só você.

A clareza dói antes de ajudar.
```

---

### SLIDE 4 / FRAMEWORK 1

```
TÍTULO: A primeira decisão difícil

Arquétipo.
Não "qual palavra soa bem" —
mas qual padrão inconsciente você quer ativar.

Eu me via como Criador. Minha marca sinalizava Sábio.
Eram diferentes o suficiente para criar ruído
em tudo que eu produzia — sem eu perceber por quê.

A primeira decisão foi alinhar os dois.
```

---

### SLIDE 5 / FRAMEWORK 2

```
TÍTULO: O anti-posicionamento é mais difícil

Definir o que você é — relativamente fácil.
Definir o que você explicitamente não é — cirúrgico.

Meu anti-posicionamento inclui:
não ser "coach de produtividade"
não falar para quem quer atalho
não simplificar para parecer acessível

Cada "não" que entrou no documento
foi uma identidade que eu havia assumido sem perceber.
```

---

### SLIDE 6 / FRAMEWORK 3

```
TÍTULO: O que um brand book faz na prática

Depois de pronto, cada decisão de conteúdo
ficou mais rápida — não porque ficou mais fácil,
mas porque o critério estava definido.

"Esse tema está alinhado com algum dos 5 pilares?"
"Esse gancho quebra o vocabulário da marca?"
"Essa frase tem subtexto ou só afirmação?"

O documento não cria a marca.
Ele externaliza o raciocínio que você já fazia internamente —
mas de forma inconsistente.
```

---

### SLIDE 7 / RESPIRO POÉTICO

```
"Clareza sobre si mesmo
é o ativo mais subvalorizado do criador."
```

*Teste: ✅ Citável isolado*

---

### SLIDE 8 / SÍNTESE

```
TÍTULO: O que mudou depois

Construir em público não é só mostrar processo.
É tornar o processo verificável — por você mesmo.

Quando você documenta o que está fazendo e por quê,
você cria um espelho que não deixa você se enganar
sobre a distância entre o que você declara e o que entrega.

Isso é incômodo. E é exatamente o ponto.
```

---

### SLIDE 9 / FECHAMENTO

```
TÍTULO: Uma pergunta direta

Se você tivesse que escrever
em uma linha — sem rodeios —
quem exatamente você atende e o que muda para essa pessoa:

Você conseguiria?
```

---

## LEGENDA

```
Passei a semana construindo meu próprio brand book
usando o mesmo processo que uso com clientes.

O que ninguém te conta: é mais difícil fazer o seu.
Porque você não tem a distância do cliente — tem o ego do dono.

Se você tivesse que escrever em uma linha quem você serve, conseguiria?
```
