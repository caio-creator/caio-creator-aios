# P06 — Pilares de Conteúdo

**Fase:** 6 de 9
**Objetivo:** Definir os 4-5 pilares editoriais que estruturam toda a produção de conteúdo da marca

---

## Quando Usar

Use após completar a narrativa (Fase 5). Os pilares de conteúdo são os temas permanentes da marca — não categorias de produto, mas territórios de conversa que atraem, educam e convertem.

---

## Prompt Principal

```
Você é um Diretor Editorial de Conteúdo especializado em estratégia de marca. Com base em tudo que sabemos sobre [NOME_DA_MARCA], defina os pilares editoriais e a estrutura de conteúdo.

━━━ CONTEXTO ━━━
[INSERIR POSICIONAMENTO + PERSONAS + VOZ + NARRATIVA]

━━━ DEFINIÇÃO DOS PILARES ━━━

**O QUE É UM PILAR EDITORIAL?**
Um pilar é um tema de conversa que:
- A marca tem autoridade genuína para falar
- O público-alvo tem interesse real em consumir
- Alinha com o posicionamento e os valores da marca
- Pode ser sustentado por meses ou anos com conteúdo fresco

Não confunda pilar com categoria de produto. "Nosso serviço X" não é um pilar.
"Como [problema que o serviço resolve] afeta [vida do cliente]" pode ser um pilar.

**ESTRUTURA DE CADA PILAR:**

Para [NOME_DA_MARCA], crie 4-5 pilares. Para cada um:

**PILAR [N]: [Nome do Pilar]**

- **Definição:** O que este pilar cobre — o território de conversa
- **Por que a marca tem autoridade aqui:** Conexão com o briefing
- **Por que o público se importa:** Ligação com as dores/aspirações da persona
- **Objetivo estratégico:** Awareness, consideração, conversão, ou retenção?
- **Tom específico neste pilar:** Mais próximo de qual ponto das escalas de voz?

**10 Tópicos Iniciais:**
[lista de 10 temas específicos que cabem neste pilar]

**Prompt Gerador para este Pilar:**
Um prompt que qualquer IA pode usar para gerar conteúdo neste pilar:
```
Escreva um [formato] sobre [tópico] para [NOME_DA_MARCA], seguindo estas diretrizes:
- Público: [persona]
- Objetivo: [awareness/consideração/conversão]
- Tom: [especificação]
- Evitar: [armadilhas]
```

━━━ MIX DE CONTEÚDO ━━━

Defina a proporção ideal de cada pilar na produção mensal:
- Pilar 1: ___% do conteúdo
- Pilar 2: ___% do conteúdo
- Pilar 3: ___% do conteúdo
- Pilar 4: ___% do conteúdo
- Pilar 5 (se houver): ___% do conteúdo

Justifique o mix baseado nos objetivos de negócio e estágio do cliente na jornada.

━━━ MIX DE FORMATOS ━━━

Quais formatos a marca priorizará e em qual proporção?

| Formato | Proporção | Pilar principal | Objetivo |
|---------|-----------|----------------|---------|
| Post estático | ___% | | |
| Carrossel | ___% | | |
| Reels/TikTok | ___% | | |
| Stories | ___% | | |
| Artigo/texto longo | ___% | | |
| E-mail | ___% | | |

━━━ RÉGUA DE CONTEÚDO ━━━

Defina a distribuição entre tipos de conteúdo:

| Tipo | Proporção | Exemplo |
|------|-----------|---------|
| Conteúdo Educativo | ___% | "Como fazer X" |
| Conteúdo de Valor/Inspiração | ___% | "Por que X importa" |
| Conteúdo de Prova Social | ___% | Depoimentos, resultados |
| Conteúdo de Bastidor | ___% | Processo, equipe |
| Conteúdo de Venda | ___% | Ofertas, CTA direto |

━━━ ANTI-PILARES ━━━

3 temas que a marca definitivamente NÃO abordará — e por quê:
1. [tema + motivo]
2. [tema + motivo]
3. [tema + motivo]
```

---

## Prompt de Refinamento — Validar Diferenciação

```
Os pilares que definimos podem ser usados por qualquer concorrente de [NOME_DA_MARCA]?

Para cada pilar, teste:
1. Um concorrente direto poderia ter esse pilar sem modificações? Se sim, refine.
2. Há algo no posicionamento ou diferencial de [NOME_DA_MARCA] que tornaria este pilar único?
3. O que torna este pilar distintivo — o tema, o ângulo, o ponto de vista, ou o formato?

Reescreva os pilares que não passarem no teste com o ângulo único da marca.
```

---

## Output Esperado

Seção no `editorial.md`:

```markdown
## Pilares Editoriais

### Pilar 1: [Nome]
[definição + autoridade + objetivo + 10 tópicos + prompt gerador]

[repetir para cada pilar]

### Mix de Conteúdo
[proporções por pilar e por formato]

### Régua de Conteúdo
[distribuição por tipo]

### Anti-Pilares
[o que não abordar]
```
