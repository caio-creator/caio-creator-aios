# P06 — Linha Editorial Semanal

**Fase:** 6 de 9
**Objetivo:** Criar a grade editorial semanal — o ritmo de publicação com slot, objetivo e prompt pronto para cada dia

---

## Quando Usar

Use após definir os pilares (P06-content-pillars). A linha editorial semanal transforma a estratégia em rotina executável — cada slot tem propósito claro e prompt pronto para geração de conteúdo.

---

## Prompt Principal

```
Você é um Editor-Chefe de Conteúdo Digital. Com base nos pilares e na estratégia de [NOME_DA_MARCA], crie a Linha Editorial Semanal completa.

━━━ CONTEXTO ━━━
[INSERIR PILARES + MIX DE CONTEÚDO + PERSONA + VOZ]

━━━ LINHA EDITORIAL SEMANAL ━━━

A linha editorial define: em qual dia, qual tipo de conteúdo, com qual objetivo emocional, em qual formato — e inclui o prompt de geração pronto para uso.

Crie a grade para uma semana típica de [NOME_DA_MARCA].
Defina quantos posts/semana fazem sentido para o estágio da marca.

Para cada slot da semana:

**[DIA] — Slot [N]: [Nome do Slot]**
- **Pilar:** [qual dos pilares]
- **Objetivo:** [awareness / consideração / conversão / retenção / relacionamento]
- **Formato:** [post estático / carrossel / reels / stories / texto longo]
- **Objetivo emocional:** [como o cliente deve se sentir ao consumir]
- **Tipo de conteúdo:** [educativo / inspiracional / prova social / bastidor / venda]
- **Estrutura típica:** [como o conteúdo é estruturado neste slot]
- **Prompt gerador pronto:**

```
[Prompt completo que pode ser usado para gerar conteúdo neste slot específico,
já com o tom, o objetivo, o formato e as instruções de marca da [NOME_DA_MARCA]]
```

━━━ EXEMPLO DE SEMANA COMPLETA ━━━

| Dia | Slot | Pilar | Formato | Objetivo |
|-----|------|-------|---------|----------|

━━━ GUIA DE QUALIDADE DO CONTEÚDO ━━━

**8 Critérios de Aprovação**

Todo conteúdo publicado por [NOME_DA_MARCA] deve passar nestes critérios:
1. [Critério específico 1 — ex: "O primeiro parágrafo cria curiosidade suficiente para ler o resto?"]
2. [Critério específico 2]
3. [Critério específico 3]
4. [Critério específico 4]
5. [Critério específico 5]
6. [Critério específico 6]
7. [Critério específico 7]
8. [Critério específico 8]

**Anti-exemplos**

3 exemplos de conteúdo que [NOME_DA_MARCA] NUNCA publicaria, com explicação de por quê:
1. [anti-exemplo + explicação]
2. [anti-exemplo + explicação]
3. [anti-exemplo + explicação]

**Checklist Pré-Publicação**

Para cada conteúdo antes de publicar:
- [ ] Tom de voz consistente com o DNA?
- [ ] Está dentro de um pilar definido?
- [ ] Objetivo claro (awareness/conversão/relacionamento)?
- [ ] CTA ou próximo passo indicado?
- [ ] Passou nos 8 critérios de qualidade?

━━━ SAZONALIDADE E DATAS RELEVANTES ━━━

Para [SETOR] e [PERSONA], quais são:
- As 5 datas mais relevantes do ano que merecem conteúdo especial?
- Os 3 momentos do ano em que o público está mais receptivo a comprar?
- As 3 épocas do ano que devem ser evitadas para lançamentos?
```

---

## Prompt de Refinamento — Prompts de Geração em Massa

```
Para o slot de [DIA] — [NOME DO SLOT], gere 4 versões diferentes do prompt de geração:

**Versão A:** Tom mais educativo (foco em ensinar)
**Versão B:** Tom mais inspiracional (foco em motivar)
**Versão C:** Tom mais prático (foco em ação imediata)
**Versão D:** Tom mais narrativo (foco em história)

Cada versão deve produzir conteúdo diferente mas igualmente fiel ao DNA de [NOME_DA_MARCA].
```
