# P08 — Diretrizes E-mail Marketing

**Fase:** 8 de 9
**Objetivo:** Definir a estratégia e estrutura completa de e-mail marketing para a marca

---

## Quando Usar

Use quando [NOME_DA_MARCA] tem ou quer construir uma lista de e-mail. O e-mail é o canal com maior ROI para marcas com relacionamento contínuo com o cliente — é o único canal que a marca controla completamente.

---

## Prompt Principal

```
Você é um Especialista em E-mail Marketing e CRM. Com base em toda a estratégia de [NOME_DA_MARCA], crie as Diretrizes Completas de E-mail Marketing.

━━━ CONTEXTO ━━━
[INSERIR POSICIONAMENTO + PERSONAS + VOZ + JORNADA DO CLIENTE]

━━━ ESTRATÉGIA DE E-MAIL ━━━

**PARTE 1: TIPOS DE E-MAIL**

Quais tipos de e-mail [NOME_DA_MARCA] deve enviar e com qual frequência?

*Newsletter/Conteúdo:*
- Frequência: [semanal / quinzenal / mensal]
- Propósito: [educar, inspirar, conectar]
- Duração ideal de leitura: [2-3 min / 5 min / 10 min]
- Formato: [texto puro / com imagens / híbrido]

*E-mail de Relacionamento (triggered):*
- Boas-vindas: enviado quando? Com qual objetivo?
- Aniversário: sim/não?
- Reengajamento: quando reengajar inativos?

*E-mail Comercial:*
- Frequência de ofertas: [x por mês máximo]
- Ratio conteúdo × venda: [ex: 4 e-mails de conteúdo para 1 de venda]
- Tom em e-mails de venda: [diferente ou igual ao de conteúdo?]

*Automações:*
- Quais sequências automáticas fazem sentido para [NOME_DA_MARCA]?
  □ Sequência de boas-vindas (quantos e-mails? Em quantos dias?)
  □ Sequência pós-compra
  □ Sequência de reengajamento
  □ Sequência de abandono de carrinho (se e-commerce)

**PARTE 2: ESTRUTURA DE E-MAIL**

Para a newsletter/e-mail de conteúdo, defina a estrutura padrão:

*Subject Line (assunto):*
- Comprimento: [menos de 50 / 50-70 / mais de 70 caracteres]
- Tom: [curioso / direto / provocador / íntimo]
- Uso de nome: personalizar com [Primeiro Nome]?
- Emojis: sim/não/quando?
- 3 fórmulas que funcionam para [NOME_DA_MARCA]:
  1. [fórmula + exemplo]
  2. [fórmula + exemplo]
  3. [fórmula + exemplo]

*Preview Text (pré-header):*
- Comprimento ideal
- Como complementa o assunto sem repetir

*Corpo do E-mail:*
- Saudação: como começa? [Oi [Nome] / Olá / Apenas conteúdo]
- Abertura: [imediata no conteúdo / breve contexto pessoal / pergunta]
- Desenvolvimento: [um tópico por e-mail ou múltiplos links]
- Parágrafos: [curtos de 2-3 linhas / médios / longos]
- Uso de imagens: [muitas / poucas / texto puro]
- Link principal: onde fica? [início / meio / fim]

*Fechamento:*
- Tom: [formal / caloroso / divertido]
- Assinatura: [nome / cargo / foto / link]
- P.S.: a marca usa P.S.? (são os trechos mais lidos nos e-mails)

**PARTE 3: TOM ESPECÍFICO PARA E-MAIL**

O e-mail é o canal mais pessoal — deve soar como carta de amigo, não comunicado de empresa.

Para [NOME_DA_MARCA]:
- A voz de e-mail é [igual / ligeiramente diferente] da voz das redes sociais
- Nível de intimidade: [muito próximo / próximo / profissional]
- Uso de histórias pessoais: [sim / raramente / não]
- Referências a situações cotidianas: [quando e quais]

**PARTE 4: SEGMENTAÇÃO**

Como [NOME_DA_MARCA] deve segmentar a lista:
- Segmentos mínimos necessários: [list de segmentos]
- O que cada segmento recebe diferente: [o que personalizar]
- Tags importantes para rastrear: [comportamentos que merecem tag]

**PARTE 5: MÉTRICAS**

KPIs prioritários:
- Taxa de abertura (benchmark do setor de [SETOR]: aproximadamente [X]%)
- Taxa de clique
- Taxa de conversão (quando aplicável)
- Taxa de cancelamento (sinal de alarme acima de [X]%)

O que NÃO medir como sucesso de e-mail:
- Tamanho da lista (vs. engajamento da lista)
```

## Prompt de Refinamento — Sequência de Boas-Vindas

```
Escreva a sequência de boas-vindas de [NOME_DA_MARCA] em 3 e-mails:

**E-mail 1 (imediato após cadastro):**
- Assunto: [proposta]
- Objetivo: [acolher + entregar o prometido + estabelecer expectativa]
- Estrutura: [como deve ser]

**E-mail 2 (3 dias depois):**
- Assunto: [proposta]
- Objetivo: [aprofundar a conexão + educar]
- Estrutura: [como deve ser]

**E-mail 3 (7 dias depois):**
- Assunto: [proposta]
- Objetivo: [converter ou qualificar a relação]
- Estrutura: [como deve ser]

Escreva em tom alinhado com o DNA de voz de [NOME_DA_MARCA].
```
