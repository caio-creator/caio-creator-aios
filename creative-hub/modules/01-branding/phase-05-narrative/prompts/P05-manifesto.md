# P05 — Manifesto de Marca

**Fase:** 5 de 9
**Objetivo:** Construir o texto de declaração de valores e visão da marca — o manifesto que inspira clientes, equipe e parceiros

---

## Quando Usar

Use após criar a Brand Story. O manifesto é o texto mais poético e apaixonado que a marca produz — não explica o que faz, declara em que acredita e por que existe.

---

## Prompt Principal

```
Você é um Redator de Manifestos de Marca — um especialista em textos que inspiram sem serem vagos, que provocam sem serem agressivos, que declaram crenças sem virarem clichê.

Com base em tudo que sabemos sobre [NOME_DA_MARCA], escreva o Manifesto de Marca.

━━━ CONTEXTO ━━━
[INSERIR BRAND STORY + POSICIONAMENTO + VOZ]

━━━ ESTRUTURA DO MANIFESTO ━━━

Um Manifesto de Marca poderoso tem 5 movimentos:

**MOVIMENTO 1: A DECLARAÇÃO DE MUNDO (Abertura)**
Começa com uma observação sobre o mundo, o setor, o momento — não sobre a marca.
Deve ser uma verdade que qualquer pessoa do público-alvo reconhece instantaneamente.
Tom: visionário, levemente provocador.

**MOVIMENTO 2: O QUE ACREDITAMOS (Crenças)**
3-5 crenças centrais da marca — cada uma começando com "Acreditamos que..." ou "Para nós,..."
Cada crença deve ser genuína (baseada em evidência do briefing) e capaz de dividir opiniões.
Uma crença que todo mundo concorda não é uma crença — é um lugar-comum.

**MOVIMENTO 3: O QUE RECUSAMOS (Heresia)**
O que a marca se recusa a aceitar — o status quo, a norma do setor, o "jeito que sempre foi feito".
Deve nomear o inimigo sem nomear concorrentes.
Tom: corajoso, não raivoso.

**MOVIMENTO 4: O QUE FAZEMOS (Compromisso)**
Como a marca age a partir das suas crenças — não produtos, mas comportamentos e compromissos.
"É por isso que nós..." → ações concretas que provam as crenças.

**MOVIMENTO 5: O CONVITE (Fechamento)**
Um convite ao leitor para fazer parte — não uma venda, mas um alinhamento de valores.
"Se você também acredita que..." → o cliente que comparte esses valores é bem-vindo.
Tom: inclusivo para o ICP, excludente para o anti-ICP.

━━━ REQUISITOS DO MANIFESTO ━━━

- Extensão: 200-350 palavras
- Tom: alinhado com o DNA de voz, mas elevado — o manifesto é o texto mais aspiracional da marca
- Evitar: clichês de marketing, promessas de features, auto-elogio
- Testar: Alguém que não conhece a marca consegue entender o que ela defende?

━━━ VERSÃO VISUAL (para design) ━━━

Depois do manifesto completo, crie uma versão fragmentada para design:
- 5-7 declarações curtas (5-10 palavras cada)
- Cada uma captura uma crença ou compromisso central
- Formato: pode virar pôster, slide, post, tatuagem metafórica

Exemplos de formato:
"Feito para durar. Não para impressionar."
"Simplicidade não é falta de ambição. É respeito pelo seu tempo."
```

---

## Prompt de Refinamento — Testar Autenticidade

```
Quero testar se o manifesto é genuíno ou performático.

Para cada crença declarada no manifesto, me dê:
1. Uma EVIDÊNCIA do briefing que prova que a marca realmente acredita nisso
2. Uma CONSEQUÊNCIA: algo que a marca abre mão ou recusa fazer por causa dessa crença
3. Um RISCO: o que acontece se a marca agir contrário a essa crença

Se não houver evidência, consequência e risco — a crença é vazia. Reescreva para refletir o que a marca realmente faz, não o que quer parecer.
```

## Prompt de Refinamento — Energia do Manifesto

```
O manifesto está correto mas está morto. Precisa de energia.

Reescreva usando estas técnicas:
1. Comece frases curtas com verbos no imperativo: "Escolha.", "Recuse.", "Construa."
2. Use perguntas retóricas que ninguém na audiência pode responder "não"
3. Contraste: "Não é sobre X. É sobre Y."
4. Anáfora: comece 3 frases seguidas com a mesma palavra
5. Termine cada movimento com a frase mais forte, não a mais segura

O manifesto deve fazer o leitor querer fazer algo — não apenas concordar.
```

- [ ] Crenças capazes de dividir opiniões (não são consenso)
- [ ] "Heresia" nomeada com coragem mas sem agressividade
- [ ] Convite final alinhado com o ICP
- [ ] Versão visual com 5-7 declarações acionáveis
