## Prompt de Refinamento — Síntese Executiva

Use quando a síntese estiver genérica demais:

```
A síntese executiva do perfil de [NOME DA REFERÊNCIA] ficou genérica.
Reescreva com estes critérios:

1. A primeira frase deve capturar O QUE É ÚNICO nesta referência (não apenas quem ela é)
2. Deve mencionar pelo menos 1 posição contraintuitiva específica
3. Deve dar ao leitor uma razão clara para estudar este modelo mental
4. Deve ser honesta sobre limitações
5. Comprimento: 5-6 frases densas, sem generalizações

Teste: Se você substituísse o nome desta referência por outro pensador do mesmo campo,
a síntese ainda faria sentido? Se sim, ela não é específica o suficiente. Reescreva.
```

---

## Critérios de Qualidade

- [ ] Síntese executiva é específica (não funciona com outro nome no lugar)
- [ ] Modelo de mundo tem pelo menos 5 premissas com implicações
- [ ] Pelo menos 5 frameworks com mecanismo e frase-síntese
- [ ] DNA de voz inclui escalas e marcas de linguagem
- [ ] Posições contraintuitivas são genuinamente contraintuitivas (não consenso)
- [ ] Limitações e pontos cegos são honestos e específicos
- [ ] Prompt de ativação compacto está pronto para uso imediato
- [ ] Template `second-brain-profile.md` preenchido e salvo
