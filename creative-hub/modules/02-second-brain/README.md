# MÓDULO 02 — SEGUNDO CÉREBRO
> Método de clonagem de modelos mentais e voz de referências estratégicas

---

## O Que É Este Módulo

O **Segundo Cérebro** é um método sistemático para extrair, estruturar e ativar o modelo mental de pensadores, líderes e referências que você quer incorporar ao seu repertório criativo e estratégico.

**Não é sobre imitar.** É sobre desenvolver a capacidade de pensar COM a perspectiva de alguém que você admira, usando os frameworks e princípios que essa pessoa usa — e eventualmente sintetizando uma voz própria a partir de múltiplas influências.

---

## Para Que Serve

- **Desenvolver repertório**: Absorver frameworks de grandes pensadores de forma estruturada
- **Criar conteúdo na voz certa**: Usar a perspectiva de uma referência para criar conteúdo mais rico
- **Simular decisões**: "O que [Referência] faria nesta situação?"
- **Enriquecer sua marca pessoal**: Combinar múltiplos modelos mentais em voz própria
- **Briefar IAs**: Criar perfis que instrucionalizam o estilo de raciocínio para uso em prompts

## Fluxo do Método

```
[SB01] Research      → Coletar tudo sobre a referência
      ↓
[SB02] Frameworks    → Extrair modelos mentais e ferramentas de pensamento
      ↓
[SB03] Voz           → Mapear padrões de linguagem, ritmo, metáforas
      ↓
[SB04] Crenças       → Identificar o sistema de crenças fundacionais
      ↓
[SB05] Compilação    → Sintetizar em perfil estruturado e ativável
      ↓
[SB06] Ativação      → Criar system prompt para usar a persona
      ↓
[SB07] Simulação     → Testar e refinar com perguntas específicas
```

**Tempo estimado por referência:** 3-6 horas de pesquisa + 2-3 horas de extração

---

## Curadoria de Referências

Antes de clonar, avalie a referência:

| Critério | Pergunta |
|----------|---------|
| **Profundidade** | Esta pessoa tem um corpo de trabalho consistente e substancial? |
| **Relevância** | Os frameworks dela se aplicam ao que você quer construir? |
| **Diferenciação** | Ela pensa de forma genuinamente diferente do status quo? |
| **Acesso** | Há material suficiente para fazer a pesquisa (livros, entrevistas, artigos)? |

**Para curadoria de referências por domínio, consulte:**
`creative-hub/data/second-brain-references.md`

## Como Usar em Projetos de Branding

O Segundo Cérebro se integra ao **Módulo de Branding** nestas fases:

- **Fase 4 (Voz)**: Referenciar pensadores com voz similar para calibrar tom
- **Fase 5 (Narrativa)**: Usar estruturas narrativas de referências para enriquecer histórias
- **Fase 6 (Editorial)**: Aplicar frameworks de conteúdo de referências na linha editorial

---

## Uso com @brand-director

```
@brand-director → *clone-reference [NOME DA REFERÊNCIA]
```

O agente executará o fluxo completo SB01→SB07 e salvará o perfil em `clients/[slug]/second-brain/[referência]/`.

---

## Ética de Uso

- Perfis clonados são ferramentas de **aprendizado e inspiração**, não de plágio
- Nunca publique conteúdo afirmando ser "escrito por" a referência
- Use para desenvolver seu próprio repertório, não para substituir a voz original
- Cite influências quando relevante — transparência é parte da autenticidade
